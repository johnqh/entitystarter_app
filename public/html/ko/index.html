Starter App

This is a template project demonstrating the Sudobility architecture pattern. It shows how to build a full-stack application with shared types, a Hono API, TanStack Query hooks, Zustand stores, and a React web app.

View DocumentationView Histories

Shared Types

TypeScript type definitions shared between frontend and backend for type safety across the stack.

TanStack Query

Automatic caching, cache invalidation, and background refetching with TanStack Query hooks.

Zustand Stores

Lightweight state management with Zustand for offline cache and business logic.
